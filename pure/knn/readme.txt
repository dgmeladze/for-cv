Задача.
По сути это "зачаток" для блока "с этим товаром покупают" или "вам может понравиться" для различных страниц.
Можно так же использовать бОльшую часть скрипта для такого блока в корзине товаров,
который ориентировался бы на товары из корзины.

Описание
Что имеется, если вкратце
БД с 3 таблицами
product - id и название товара
customers - id и имя
order_product - id товара, id пользователя и "сколько раз куплен товар".

БД наполнял с помощи фабрики.
Я подобавлял еще даты во всех таблицах. Так как изначально хотел создать 3х факторный отбор: какие товары куплены,
количество раз, когда. То есть, чтобы был бы более эффективный с точки зрения маркетинга продукт. Но пока не успел.

Для демонстрации самой базы данных я сделал вывод всех нужных данных на странице демонстрации. Ниже находится форма,
которая отправляет данные в контроллер, а потом выводит в соответствии с полученными данными.
